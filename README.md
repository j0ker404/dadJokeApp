# punny

## Project setup
```
npm install
```
## Set up firebade
Create a firebaseConfig.js in the src directory
```
import firebase from "firebase";

const firebaseConfig = {
  apiKey: "",
  authDomain: "",
  databaseURL: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
  measurementId: ""
};

// initialize Firebase
export default firebase.initializeApp(firebaseConfig);

```
